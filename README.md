# Intermediate Software Development Activity 4

This activity will help to reinforce learning of the Module 4 concepts of:

- Event Handling
- Signals
- Slots
- PySide6

## Author

[Sukhpreet Kaur]

## Additional Information

[This project applies key concepts from Module 4, including:
1. **Event Handling**: Button click events are handled to facilitate adding and removing contacts.
2. **Signals and Slots**: PySide6's signals and slots are used to link button interactions
                          with their corresponding handler methods.
3. **PySide6 Widgets**: Various widgets from PySide6, such as QTableWidget,
                        QLineEdit, QPushButton, and QMessageBox,
                        are integrated to build the user interface.]
